# Apple Root CA Certificates

Download these certificate files from https://www.apple.com/certificateauthority/ and place them here:

- AppleIncRootCertificate.cer (Apple Inc. Root)
- AppleRootCA-G2.cer (Apple Root CA - G2 Root)
- AppleRootCA-G3.cer (Apple Root CA - G3 Root)

These are required for verifying Apple In-App Purchase transactions.
